Na página "Funcionarios":
Esta aplicação demonstra o uso de Angular com dados de funcionarios (id, nome, foto, email e subobjetos).

Já na página "Cadastro":
Os dados dos funcionários são adquiridos por uma requisição na uri 'https://bu.furb.br/mcardoso/progWeb/apiRestAval.php/cadastro/', também foi inclementado ao o usuário a capacidade de consultar, excluir e alterar funcionários.

Usando arquitetura MVC na aplicação.
